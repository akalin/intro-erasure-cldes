Example 1: Carry-less addition

Let a = 23 and b = 54. Then, with carry-less arithmetic, 
  a = 23 =  10111b
^ b = 54 = 110110b
           -------
           100001b
so a ⊕ b = 100001b = 33.

Example 2: Carry-less multiplication

Let a = 23 and b = 54. Then, with carry-less arithmetic, 
   a = 23 =       10111b
^* b = 54 =      110110b
            ------------
                 10111
          ^     10111
          ^   10111
          ^  10111
            ------------
             1111100010b
so a ⊗ b = 1111100010b = 994.
